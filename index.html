    Minha Motivação

    A minha motivação para criar esse site foi criar um lugar onde meus amigos de programção pudedssem fazer uma consulta rápida no código dos cards, além de testar experimentar minhas habilidades em programação e dar uma viajada no que eu consigo fazer.

    Sobre o Material

    O materia que serviu de inspiração pra esse curso foi do curso de desenvolvimento web da Cod3r que você pode conhercer Acessando Aqui Mas todos os cards foram desenvolvidos por mim, baseados nas aulas do curso da plataforma cod3r.

    Meu GitHub

    Se você achou esse projeto minimamente interessante e quer descobrir o código por trás dele, de uma passada no meu Github lá tem outros projetos que eu tenho certeza que você vai curtir também, não esquece de curtir e seguir meu perfil.
